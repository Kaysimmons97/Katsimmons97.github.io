About Us

KAC Pharmacy was designed by Katherine Simmons (UX/UI Designer), Clarence Cloud (Database Developer), and Alexander Stott (Framework Developer). What is UX/UI design? User interface design or user interface engineering is the design of user interfaces for machines and software, such as computers, home appliances, mobile devices, and other electronic devices, with the focus on maximizing usability and the user experience. What is database design? Database design is the organization of data according to a database model. The designer determines what data must be stored and how the data elements interrelate. With this information, they can begin to fit the data to the database model. Database management system manages the data accordingly. What is a framework? A framework in programming is a tool that provides ready-made components or solutions that are customized in order to speed up development. A framework may include a library, but is defined by the principle of inversion control(IoC) What is Web2py? Web2py is an open-source web application framework written in the Python programming language. Web2py allows web developers to program dynamic web content using Python. As students of the 14th cohort of the Cleveland Codes Software Developers Program. The KAC Pharmacy web application is the visual representation of the technical skills that were newly acquired or solidified throughout this 14-week intensive program. For this capstone we were split into a three-person team and tasked to create a Customer Relationship Management System. Customer relationship management is a process in which a business or other organization administers its interactions with customers. These systems typically utilize data analysis to study and create a targeted user experience based on large amounts of information.

Our KAC Pharmacy CRM based on the concept of a mock mail order dose-pack distribution pharmacy similar to PillPack, a simplified pharmacy solution provided through Amazon Pharmacy. We achieved our CRM model through the utilization of Python, JavaScript/JQuery, Bootstrap, SQLi, Figma , and Mockaroo . The KAC Pharmacy team made full use of GitHub and Visual Studio Code to manage the creation and modification of code being created for out project. Having organized all of our files in the shared repository, we were then able to transfer the code into the web2py framework, which was required for out capstone presentation. Our SQL databases were inclusive of random data created via the Mockaroo app and hardcoded data used created during the UX/UI design portion of our project. This data was read via SQlite into the web2py database framework.
