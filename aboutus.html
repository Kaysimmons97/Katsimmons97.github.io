As students of the 14th cohort of the Cleveland Codes Software Developers Program. The KAC Pharmacy web application is the visual representation of the technical skills newly acquired or solidified throughout this 14-week intensive program. For this capstone, we were split into a three-person team and tasked to create a Customer Relationship Management System. Customer relationship management is a process in which a business or other organization administers its interactions with customers. These systems typically use data analysis to study and create a targeted user experience based on large amounts of information.


Our KAC Pharmacy CRM is based on a mock mail order dose-pack distribution pharmacy, similar to PillPack, a simplified pharmacy solution provided through Amazon Pharmacy. We achieved our CRM model through the following:

  • Python
  • JavaScript
  • JQuery UI
  • JQuery
  • Bootstrap 3 and 4 ,/li>
  • SQLite
  • Figma
  • Mockaroo ,
  • HTML5
  • CSS

    • The KAC Pharmacy development team made full use of GitHub and Visual Studio Code to manage the creation and modification of code being created for our project. Having organized all our files in the shared repository, we were then able to transfer the code into the web2py framework, which was required for our capstone presentation. Our SQL databases included random data generated via the Mockaroo app and hard-coded data used during the UX/UI design portion of our project. This data was read via SQlite into the web2py database framework.

    What does all this thechnical jargon mean?

    What is UX/UI design?

    User interface design or user interface engineering is the design of user interfaces for machines and software, such as computers, home appliances, mobile devices, and other electronic devices, with the focus on maximizing usability and the user experience.

    What is database design?

    Database design is the organization of data according to a database model. The designer determines what data must be stored and how the data elements interrelate. With this information, they can begin to fit the data to the database model. Database management system manages the data accordingly.

    What is a framework?

    A framework in programming is a tool that provides ready-made components or solutions that are customized in order to speed up development. A framework may include a library, but is defined by the principle of inversion control(IoC).

    What is Web2py?

    Web2py is an open-source web application framework written in the Python programming language. Web2py allows web developers to program dynamic web content using Python.
